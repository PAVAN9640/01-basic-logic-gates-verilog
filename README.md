# Basic Logic Gates using Verilog

## Overview
This project implements fundamental digital logic gates using Verilog HDL.

Implemented Gates:
- AND Gate
- OR Gate
- XOR Gate
- NAND Gate
- NOR Gate

## Tools Used
- Verilog HDL
- Xilinx Vivado

## Files
- logic_gates.v
- logic_gates_tb.v
- waveform.png

## Simulation Result
All input combinations (00, 01, 10, 11) were tested successfully.

## Concepts Learned
- Module Syntax
- Assign Statements
- Combinational Logic Design
- Testbench Development
- Functional Simulation
